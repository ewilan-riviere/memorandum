# Bun

{{ $frontmatter.description }}

## Install

::: code-group

```sh [Linux]
curl -fsSL https://bun.sh/install | bash
```

```ps1 [Windows]
powershell -c "irm bun.sh/install.ps1 | iex"
```

:::

Reload shell

```sh
source ~/.zshrc
```

## Usage

To use Bun, run the following command:

```sh
bun
```

To get version information:

```sh
bun --version
```
